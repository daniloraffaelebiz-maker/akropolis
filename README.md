# Akropolis Schagen — Website (GitHub Pages)

Multi-page website (HTML/CSS/JS) met:
- Kleuren: zwart #000000 + goud #e1a730
- Fonts: Libre Baskerville & Tenor Sans
- Logo: `assets/logo-akropolis.png`
- Menu PDF: `assets/akropolis-dinerkaart.pdf`

## In 1 keer live zetten op GitHub Pages
1. Maak een repo (bijv. `akropolis`)
2. Upload **alles** uit deze map naar de root van je repo (index.html moet bovenin staan)
3. Settings → Pages → Deploy from a branch → Branch: `main` / Folder: `/ (root)` → Save
4. Live URL: `https://<username>.github.io/akropolis/`
