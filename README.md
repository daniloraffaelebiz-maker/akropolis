# Akropolis Schagen — Website project

Dit is een statische multi-page website (HTML/CSS/JavaScript) klaar voor GitHub Pages.

## Bestanden
- `index.html`, `menu.html`, `reserveren.html`, `contact.html`
- `assets/styles.css` (zwarte + gouden styling, fonts)
- `assets/script.js`
- `assets/logo-akropolis.png`
- `assets/akropolis-dinerkaart.pdf`

## Lokale preview
Open `index.html` in je browser.
Tip: met VS Code kun je ook “Live Server” gebruiken voor een lokale preview.
