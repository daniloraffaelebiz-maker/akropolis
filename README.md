# Akropolis Schagen — Website map (HTML/CSS/JS)

Deze map is klaar voor GitHub Pages.

## Structuur
- `index.html`, `menu.html`, `reserveren.html`, `contact.html`, `404.html`
- `assets/css/styles.css`
- `assets/js/script.js`
- `assets/img/logo-akropolis.png`
- `assets/pdf/akropolis-dinerkaart.pdf`

## Live zetten op GitHub Pages
1. Maak een repo (bijv. `akropolis`)
2. Upload alle bestanden uit deze map naar de root van je repo
3. Settings → Pages → Deploy from a branch → `main` + `/(root)` → Save
